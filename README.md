# Products Catalog

E-commerce demo Android app using the [fakestoreapi.com](https://fakestoreapi.com/) APIs.

## Used: 
* Kotlin
* MVVM.
* Clean Architecture.
* Modularization.
* Jetpack Navigation Component.
* Data binding.
* Coroutines.
* Material 3.
* Gradle KTS with Version Catalogs.
* Unit testing.
